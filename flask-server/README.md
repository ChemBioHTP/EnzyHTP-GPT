# EnzyHTP-GPT Flask Server

This directory contains the Python Flask backend for the EnzyHTP-GPT application.

## OpenAI Runtime

- Runtime switch: set `OPENAI_RUNTIME=assistants` or `OPENAI_RUNTIME=responses` in environment variables.
- Recommended SDK baseline for responses migration: `openai==2.29.0`.

For detailed information on setting up, configuring, and running the backend, please see the comprehensive documentation in the main `/docs` directory:
